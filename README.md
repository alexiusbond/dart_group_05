## 🚀 Запуск проекта

1. Установите Dart SDK  
   → Для Windows См. [docs/dart_installation_win.md](docs/dart_installation_win.md)
   
   → Для MacOs См. [docs/dart_installation_mac.md](docs/dart_installation_mac.md)

2. Клонируйте репозиторий:
   ```bash
   git clone <url вашего репозитория>
   cd <имя_проекта>
   ```

3. Установите зависимости:
   ```bash
   dart pub get
   ```

4. Запустите приложение:
   ```bash
   dart run
   ```

## 🧭 Полезные команды Dart

| Команда | Назначение |
|----------|-------------|
| `dart run` | Запуск программы |
| `dart format .` | Форматирование кода |
| `dart pub add <package>` | Добавить зависимость |
| `dart pub get` | Установить зависимости |
| `dart test` | Запустить тесты |
| `dart compile exe bin/main.dart` | Сборка исполняемого файла |

## 💡 Советы по работе в VS Code

- **Форматирование:** `Shift + Alt + F`  
- **Дублирование строки:** `Shift + Alt + ↓`  
- **Перемещение строки:** `Alt + ↑ / ↓`  
- **Автоформат при сохранении:**  
  Включите параметр `Format on Save` в настройках VS Code.
